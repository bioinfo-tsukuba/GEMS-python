GEMS Python Documentation
=========================

The GEMS Python toolkit provides reusable building blocks for constructing
and simulating experiment workflows. The package centres around two
configurations:

* :mod:`gems_python.one_machine_problem_interval_task` for single-resource
  execution models.
* :mod:`gems_python.multi_machine_problem_interval_task` for
  multi-resource scheduling with machine allocation.

Quick start instructions are available in the project ``README.md``.
This Sphinx site hosts detailed API-level documentation and workflow guides.
